Les bases HTML

Première approche

'go sur l'exemple d'ancre'

L'html est un langage non pas de programmation mais de balises. Chacune de ses balises est soit fonctionnelle par paire, soit auto-fermante.
Il est important d'indenter son code afin de pouvoir s'y retrouver, surtout quand des divs se retrouvene tdans d'autres (raccourci clavier: Alt + Shift + F
Chaque balise peut être distinguées de ses collègues par une classe ou un ID. Ces derniers permettent d'appeler son contenu ou son style avec d'autres pages, ou directmeent le css:
Exemple: img src="lien.fr" alt="Je suis une image" class="coucou" id=salut

Structure de base

(P.S.: chaque > signifie "balise", donc doit entourer son contenu de < >)
>!DOCTYPE html
>html
>head
>title Titre dans l'entête >/title
>/head
>body
>p Paragraphe contenu >/p
>/body
>/html

Il est possible de recréer ce code automatiquement à la création d'une page:

  • ! + tab dans la barre de recherche; pour sélectionner l'entête html à la création d'un fichier;
  • Ctrl + Espace; pour pouvoir charger la structure de base (à condition que le format du fichier soit déjà enregistré en .html)

La balise head permet d'intégrer les informations annexes de la page. Seule la balise titile et son contenu seront visibles sur le navigateur.
Ce dernier permet de changer le texte affiché dans l'étiquette de "nouvel onglet" sur votre navigateur.
Le reste de la balise head va surtout réunir les documents importants pour dynamiser ou décorer l'affichage de notre page Html, en lui ajoutant des contenus comme du CSSS ou du JavaScript:

  • >meta charset="utf-8"; permet de lire chaque caractère spécial correctement, sinon il affichera ce qu'il pourra;
  • meta name="keywords" content="mes,mots,clés"; permet de référenceer la page plus facilement grâce à une liste de mots-clés;
  • meta name="description" content="description personnalisée"; permet de donner une description personnalisée lorsque le navigateur demandera une recherche du site;
  • link rel="stylesheet" href="main.css"; signifie que le navigateur doit rechercher un fichier stylesheet ayant comme nom de document main.css pour afficher la page avec un certain style contenu dans ce fichier;
  • script type="text/javascript" src="chemin/vers/mon/fichier/javascript.js"; permet au navigateur de rajouter des scripts JavaScript sur la page, en utilisant le lien fourni pour fonctionner. On fait appel à certains de leurs scripts dans la balise body grâce à la balise >script

P.S.: Il est conseillé de faire appel au script uniquement dans le body, car le navigateur va lire la page ligne par ligne, et s'il y a un problème dans ce script et qu'il est placé avant le contenu de la page html, le navigateur risque de planter son chargement à cause du script.

Le "body"

Il existe deux grandes classes d'élément:

  1. Block: l'élément prend toute la largeur de la ligne et chaque élément se place l'un en dessous de l'autre. Il peut néanmoins contenir plusieurs éléments inline;
  2. Inline: l'élément prend la largeur de son contenu et peut aussi contenir des éléments inline, mais pas d'élément block.

Les titres

Il est possible d'écrire plusieurs tailles de titres dans une page, les tailles varient selon si l'ont utilise un chiffre plus ou moins grand compris entre 1 et 6, précédé de la lettre h dans sa balise:

  1. Bonjour

    en h1

  2. Bonjour

    en h2

  3. Bonjour

    en h3

  4. Bonjour

    en h4
  5. Bonjour
    en h5
  6. Bonjour
    en h6

Les liens

Il est possible de faire appel à un élément extérieur à la page en lui intégrant un lien depuis un dossier ou un url:
>a href="lien.fr">Je possède un lien cliquable>/a; permet d'intégrer le lien vers lien.fr sur un texte;
>img src="../image.png"; permet d'ouvrir l'image contenue dans le dossier parent du fichier actuel, le lien présent peut changer selon l'organisation des fichiers du site.
Le premier lien est un lien absolu car il désigne un lien qui est accessible de partout avec cette adresse. Le second est dit relatif car ce chemin peut changer selon l'organisation des fichiers internes du site.

Il est même possible de créer un lien vers une création de mail:
a href="mailto:mail@contact.fr"; permet de créer un lien cliquable vers un envoi de mail à l'adresse ciblée.

Lors de l'utilisation de liens internes au site, il est possible de naviguer entre chaque fichier et dossier par ce biais:

  • .. permet de cibler le dossier qui contient la page ciblée actuellement, cela permet de revenir au dossier-parent;
  • / permet d'entrer dans le dossier visé actuellement.

Il est conseillé de toujours intégrer les attributs src et alt aux images, la première pour intégrer son lien pour être trouvée, la seconde pour permettre une lecture orale aux personnes malvoyantes, ou si le chargement de l'image a échoué.

Les tableaux

On appelle à la création et l'affichage d'un tableau par la balise table. Il est ensuite réparti en lignes:

  • On ouvre le tableau avec la balise table;
  • On indente chaque ligne du tableau une par une avec la balise tr;
  • On créé une étiquette (affichée en gras de base) avec la balise th;
  • On créé une cellule de base avec la balise td;
  • Chaque cellule du tableau peut posséder un style CSs particulier avec sa classe, ou trier par balise, ligne ou encore tableau directement.

Il est possible de fusionner plusieurs cellules d'un tableau grâce aux attributs colspan et rowspan:

  1. colspan permet de fusionner deux cellules dans le même tr, deux celulles l'une à côté de l'autre ne feront plus qu'une;
  2. rowspan permet de fusionner deux cellules l'une sous l'autre, cela peut permettre de fusionner deux lignes sur quelques cellules en les désignant avec et attribut.

Les listes

On peut créer des lites numérotées ou à puces selon la balise utilisée. ul désigne les listes à puces, tandis que ol désigne les listes numérotées.
Il est aussi possible d'indenter une liste dans une liste déjà existante:

  • Je suis première !
  • Je suis seconde !
    1. Je suis troisième !
    2. Non c'est moi !
  • Je ferme la marche ...

Les citations

Il existe quelques balises de citation: blockquote, q, abbr, address, cite et les bdo. Chacune a ses spécificités mais très peu utilisées.
On peut aussi écrire du html en commentaire en entourant le code avec !-- et -- (ne pas oublier d'ouvrir le chevron avant le ! et le fermer après la fin des --).

Les ancres

Une ancre se définit par une id d'une balise. Cette dernière permet de rediriger un lien cliquable vers lui lorsque ce lien le cible.

P.S.: Petits tips, il est possible de créer plusieurs balises, avec un id qui s'incrémente et du contenu déterminé: p#chap$*6>lorem500 par exemple:

  1. une balise paragraphe
  2. à l'id chap1 à 6
  3. en 6 exemplaires
  4. contenant 500 caractères de lorem ipsium
